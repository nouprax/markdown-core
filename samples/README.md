# Samples

The package-owned consumer projects are the executable integration samples:

- Swift: `packages/swift-markdown-core/Tests/Consumer/`
- Kotlin: `packages/kotlin-markdown-core/consumers/`
- ECMAScript: the packed-package consumer in `packages/es-markdown-core/tests/`

They stay with their owning packages so each native toolchain verifies the
same code that documents consumption. Small API snippets also appear in the
root README and each package README.
