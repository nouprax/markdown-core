# Markdown Core 1.0.1

This is the first coordinated Markdown Core release across the C, Swift,
Kotlin, and ES packages. This patch improves the Kotlin development experience
and release documentation. The public parsing and canonical AST APIs are
unchanged.

- Android Studio and IntelliJ IDEA now retain the Kotlin Multiplatform source
  sets after a Gradle sync, including common, JVM, Android, and Native sources.
- The repository provides an `All Kotlin tests` Gradle entry for running the
  complete test set supported by the current host, including both Android
  managed-device page-size configurations.
- Consumer, development, toolchain, testing, and release guidance has been
  expanded and clarified.
